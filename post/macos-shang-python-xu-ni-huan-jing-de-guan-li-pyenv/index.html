macOS 上 Python 虚拟环境的管理: pyenv

2020-10-27 1 min read # Python

"Shit moutain."

项目地址

https://github.com/pyenv/pyenv

安装

brew install pyenv

命令

列出当前已安装的 python 版本

pyenv version

列出可安装版本

pyenv install --list

列出当前可用的虚拟环境

pyenv  virtualenvs

创建虚拟环境

pyenv virtualenv <Python 版本号> <虚拟环境名称>

激活虚拟环境

pyenv activate <虚拟环境名称>

退出虚拟环境

pyenv deactivate

删除虚拟环境

pyenv uninstall <虚拟环境名称>

Vscode

把工作区配置文件里的 "python.pythonPath" 换成 pyenv which python 打印的路径。
